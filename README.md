SMARTTODOMANAGER [ANDROID STUDIO KOTLIN OTTER 2 FEATURE DROP 2025.2.2]

Smart ToDo Manager is a modern, feature-rich Android application designed to help users organize their daily tasks efficiently.
Built using the latest Android development practices, it features a sleek UI with dark mode support and local data persistence.

FEATURES

- User Authentication: Secure Login and Registration screens.
- Task Management: Create, edit, and delete tasks with ease.
- Dynamic Theming: Custom-designed Light and Dark modes using Material 3.
- Real-time Updates: Built with Jetpack Compose for a reactive and fluid user experience.
- Local Database: Offline support using Room Database to ensure your tasks are always accessible.
- Background Tasks: Integration with WorkManager for handling scheduled notifications and updates.

TECH STACK

- Language: Kotlin
- UI Framework: Jetpack Compose (Material 3)
- Architecture: MVVM (Model-View-ViewModel)
- Navigation: Jetpack Navigation Compose
- Database: Room Persistence Library
- Background Processing: WorkManager
- Networking: Retrofit & Gson (for API integration)
- Dependency Management: Gradle (Kotlin DSL)
